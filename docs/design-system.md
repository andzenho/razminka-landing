# Дизайн-код «Разминка» — Премиум-минимал

Направление: почти монохром, максимум воздуха, большая типографика, **один** сдержанный
тёплый акцент. Дорого, чисто, честно, без давления. Не похоже на «инфобиз».

Принцип: 90% страницы — графит на тёплой бумаге. Акцент (глина/терракота) появляется
редко и всегда означает «сюда» — кнопки, ключевые цифры, галочки.

---

## 1. Цвет

| Токен            | HEX       | Где                                             |
|------------------|-----------|-------------------------------------------------|
| `--paper`        | `#F6F4EF` | Основной фон страницы (тёплая бумага)            |
| `--paper-2`      | `#FBFAF7` | Чередующийся фон секций                          |
| `--surface`      | `#FFFFFF` | Карточки, плашки                                 |
| `--ink`          | `#191818` | Основной текст, заголовки (графит, не чёрный)    |
| `--ink-70`       | `#4B4A48` | Вторичный текст                                  |
| `--ink-50`       | `#7A7873` | Мета, подписи, плейсхолдеры                      |
| `--line`         | `#E7E3DA` | Волосяные границы, разделители                   |
| `--accent`       | `#C1543B` | Фон кнопок, крупные акцентные цифры              |
| `--accent-strong`| `#A5442E` | Ховер кнопок, акцентный **текст** (контраст AA)  |
| `--accent-tint`  | `#F3E5DF` | Мягкая подложка акцента (бейджи, галочки)        |
| `--ink-inv`      | `#F6F4EF` | Текст на тёмном (подвал)                         |
| `--footer-bg`    | `#191818` | Фон подвала                                      |

**Правила акцента**
- Акцент = «действие/важное». На экране одновременно максимум 1–2 акцентных объекта.
- Акцентный **текст** — только `--accent-strong` (на белом проходит контраст).
- Белый текст на `--accent` — только крупный/жирный (кнопки, ≥18px bold).
- Никаких градиентов, неона, вторых-третьих цветов. Один акцент — это фича направления.

## 2. Типографика

Пара: **серифный дисплей** для эмоции + **чистый гротеск** для чтения. Контраст = премиум.

- **Дисплей / заголовки:** `Playfair Display` (500/600/700). Кириллица есть.
- **Текст / интерфейс:** `Inter` (400/500/600). Кириллица отличная.
- Фолбэки: серив → `Georgia, "Times New Roman", serif`; гротеск → `system-ui, -apple-system, "Segoe UI", Roboto, sans-serif`.
- *В артефактах Claude внешние шрифты блокируются CSP — там работаем на фолбэках; на боевом сайте подключаем Google Fonts или самохостим.*
- *Альтернатива, если серив покажется слишком «глянцевым»: дисплей тоже на `Inter`/`Manrope` 700 с плотным трекингом. Оставляем как запасной вариант.*

**Шкала (fluid, mobile-first, через `clamp`)**

| Роль              | Размер (clamp)                    | Шрифт    | LH   | Трекинг |
|-------------------|-----------------------------------|----------|------|---------|
| Display (hero H1) | `clamp(2.5rem, 6.5vw, 4.75rem)`   | Playfair | 1.05 | -0.01em |
| H2 (секция)       | `clamp(1.75rem, 4vw, 2.75rem)`    | Playfair | 1.1  | -0.01em |
| H3 (карточка)     | `clamp(1.25rem, 2.4vw, 1.6rem)`   | Inter 600| 1.2  | -0.005em|
| Lead (подзаг.)    | `clamp(1.125rem, 2vw, 1.375rem)`  | Inter 400| 1.5  | 0       |
| Body              | `1.0625rem` (17px)                | Inter 400| 1.6  | 0       |
| Small / meta      | `0.875rem`                        | Inter 500| 1.4  | 0.02em  |
| Eyebrow (надзаг.) | `0.8125rem` UPPERCASE             | Inter 600| 1.2  | 0.12em  |
| Metric (цифра)    | `clamp(2.5rem, 6vw, 4rem)`        | Playfair | 1    | -0.02em |

**Правила**
- Ширина колонки текста ≤ ~68 символов (`--measure: 34rem`).
- Заголовки — графитом; акцент в заголовке только на 1 ключевом слове, редко.
- Не растягивать капслок на длинные строки — только eyebrow/бейджи.

## 3. Ритм и пространство (шкала 8px)

`--space-1:4 · -2:8 · -3:12 · -4:16 · -5:24 · -6:32 · -7:48 · -8:64 · -9:96 · -10:128` (px→rem)

- Вертикаль секции: `--section-y: clamp(4rem, 9vw, 7.5rem)`.
- Между заголовком секции и контентом: `--space-7` (48).
- Внутри карточки: `--space-5`/`--space-6`.
- Воздух — главный инструмент направления. Лучше меньше элементов и больше пустого.

## 4. Контейнеры и сетка

- `--container: 1120px` — общий предел контента, боковые поля `clamp(1.25rem, 5vw, 3rem)`.
- `--container-narrow: 720px` — для текстовых блоков (герой, честная часть, оффер).
- Программа: 4 карточки → 1 колонка (моб) / 2 (планшет) / 4 или 2×2 (десктоп).
- Mobile-first: базовые стили под телефон, `min-width` брейкпоинты: `48rem` (768), `64rem` (1024).

## 5. Форма, границы, тени

- Радиусы: `--radius-sm:10px · --radius:16px · --radius-lg:24px · --radius-pill:999px`.
- Карточки: `--surface` + `1px solid --line`, радиус `--radius`. Тень — почти нет.
- Тени (очень мягкие, редко): `--shadow-sm: 0 1px 2px rgba(25,24,24,.04)`,
  `--shadow-md: 0 8px 30px rgba(25,24,24,.06)`. Премиум = сдержанность, не «выпуклость».
- Разделение секций — сменой фона (`paper` ↔ `paper-2`) и волосяной линией, не тенями.

## 6. Компоненты

**Кнопки**
- Primary: фон `--accent`, текст белый, `--radius-pill`, паддинг `16px 28px`, вес 600.
  Ховер → `--accent-strong` + лёгкий подъём (`translateY(-1px)`). Фокус — видимое кольцо.
- Secondary: прозрачная, `1px solid --ink`, текст `--ink`. Для второстепенных действий.
- Ghost/ссылка-стрелка: текст `--accent-strong` + «→». Для «читать дальше».
- Размер CTA — крупный, палец попадает (мин. высота 52px).

**Format-бейдж** (в карточке дня): капслок-мини на `--accent-tint`, текст `--accent-strong`,
`--radius-pill`, мелкий. Типы: `записанный урок` · `практика с проверкой` · `прямой эфир`.

**Карточка дня**: номер дня крупно (Playfair, `--ink-50`) → бейдж формата → заголовок →
3 буллета → строка «Результат дня →». Дни 3 и 4 — акцентная рамка/фон, они ядро.

**Чек-лист результатов**: пункт = акцентная галочка (в кружке `--accent-tint`) + текст.
Крупно, с воздухом между строк.

**Метрика (автор)**: большая Playfair-цифра (акцент или графит) + подпись `--ink-50` под ней.

**Аккордеон (FAQ)**: строка-вопрос + `+`/`−`, тонкая линия снизу, плавное раскрытие.
Закрыт по умолчанию.

**Липкая кнопка (моб.)**: закреплённая снизу панель с primary-кнопкой, появляется после героя,
прячется на секции `price`.

**Заголовок секции**: eyebrow (капслок, `--ink-50`) → H2 → опц. lead. Слева или по центру.

## 7. Движение

- Дозировано: появление секций (fade + подъём 8–12px) на скролле, ховеры кнопок/карточек.
- Длительность 160–240ms, easing `cubic-bezier(.2,.6,.2,1)`.
- Уважать `prefers-reduced-motion` — отключать анимации.
- Никаких автокаруселей, прыгающих элементов, конфетти.

## 8. Do / Don't

**Do:** много воздуха · крупная типографика · один акцент · честные формулировки ·
реальное фото Карины хорошего качества · волосяные линии.

**Don't:** второй яркий цвет · градиенты/неон · фейковые таймеры и зачёркнутые цены ·
сток с грустными людьми · тяжёлые тени · капслок на длинных фразах · эмодзи-спам.
