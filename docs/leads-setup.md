# Настройка: лиды → Google Таблица

Сайт статический, поэтому лиды пишутся в Google Таблицу через **Google Apps Script**
(бесплатный веб-хук, без бэкенда и секретных ключей на нашей стороне).

## 1. Создай таблицу

1. Открой [sheets.google.com](https://sheets.google.com), создай новую таблицу.
2. Переименуй первый лист в `Leads`.
3. В первую строку добавь заголовки: `Дата`, `Имя`, `Email`, `Телефон`, `Рассылка`, `Источник`.

## 2. Добавь Apps Script

1. В таблице: **Расширения → Apps Script**.
2. Удали код-заглушку, вставь код ниже.
3. Сохрани проект (например, назови «Разминка — лиды»).

```javascript
function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Leads');
  var data = JSON.parse(e.postData.contents);

  sheet.appendRow([
    new Date(),
    data.name || '',
    data.email || '',
    data.phone || '',
    data.marketing ? 'да' : 'нет',
    data.source || ''
  ]);

  return ContentService
    .createTextOutput(JSON.stringify({ status: 'ok' }))
    .setMimeType(ContentService.MimeType.JSON);
}
```

## 3. Опубликуй как веб-приложение

1. **Развернуть → Новое развёртывание**.
2. Тип: **Веб-приложение**.
3. Выполнять от имени: **Меня**.
4. У кого есть доступ: **Все**.
5. Нажми «Развернуть», разреши доступ (это твой собственный скрипт).
6. Скопируй **URL веб-приложения** (заканчивается на `/exec`).

## 4. Подключи URL к сайту

Открой [lead-modal.jsx](../lead-modal.jsx), в самом начале файла:

```javascript
const LEADS_WEBHOOK_URL = "https://script.google.com/macros/s/REPLACE_WITH_SCRIPT_ID/exec";
```

Замени на скопированный URL. Готово — при отправке формы данные полетят в таблицу.

## Важно

- Каждое обновление кода Apps Script требует **нового развёртывания** (Развернуть →
  Управление развёртываниями → значок карандаша → Новая версия), иначе изменения не применятся.
- Запрос идёт через `navigator.sendBeacon` (или `fetch` с `no-cors` как запасной вариант) —
  это означает, что мы не можем прочитать ответ скрипта на сайте. Если лид не появился в
  таблице, проверяй логи выполнения в самом Apps Script (**Выполнения** в левом меню).
- Редирект после отправки формы — заглушка `REDIRECT_URL` в том же файле. Замени на
  реальный адрес, когда он будет готов.
