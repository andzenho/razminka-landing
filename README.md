# Лендинг «Разминка» — рабочее пространство

Трипваер-интенсив Карины (kerry.catt) перед флагманом «Большая игра». Цена 990 ₽.
Готовый дизайн собран в Claude Design и **перенесён сюда** — сайт работает локально.

## Как запустить локально

```bash
cd "Landing Kerry Katt"
python3 -m http.server 8848 --bind 127.0.0.1
# открыть http://127.0.0.1:8848/index.html
```

Нужен локальный сервер (не открывать `index.html` как файл): Babel подгружает
`lead-modal.jsx` по сети, а через `file://` это блокируется. Плюс нужен интернет —
React, Babel и шрифты грузятся с CDN.

## Архитектура (что перенесено из Claude Design)

```
index.html          # сам лендинг: React 18 + Babel-in-browser + компоненты ДС
styles.css          # точка входа стилей — @import токенов
tokens/             # дизайн-токены: fonts, colors, typography, spacing, effects, base
_ds_bundle.js       # 15 компонентов дизайн-системы (namespace DesignSystem_9e127e):
                    #   Viewfinder, TornEdge, TapedPhoto, RecIndicator, HandCircle,
                    #   HighlightMark, StatCard, Accordion, Button, TiltCard, Timecode, …
lead-modal.jsx      # форма записи (имя/почта/телефон + автоопределение страны + вебхук + редирект)
assets/             # karina.png, karina-avatar.png, reel1–8.png, verified-badge.png — реальные фото
assets/stickers/    # пиксель-арт стикеры Карины (10 шт., 5 использованы на лендинге)
```

Дизайн-система: фон `#FAFAF8`, графит `#14140F`, акценты — коралл `#FF4D3D` и лайм
`#C6F432`. Шрифты (Google Fonts): Unbounded (заголовки), Manrope (текст),
JetBrains Mono (мета), Caveat (обводки). Первый экран — статичный анимированный
фон (мягкий зелёный ореол, `.hero-bg` в `index.html`), без видео.

## Деплой

Netlify **не используется** — сайт временно живёт только в GitHub, дальше пользователь
сам загрузит его на **ГетКурс** (когда будет готов). Код: [github.com/andzenho/razminka-landing](https://github.com/andzenho/razminka-landing) (приватный).

⚠️ Перед загрузкой на ГетКурс стоит уточнить, как там принято хостить кастомный
HTML/CSS/JS-лендинг (через блок «свой код», загрузку файлов или конструктор) —
текущая структура (`index.html` + отдельные `styles.css`/`tokens/`/`_ds_bundle.js`/
`lead-modal.jsx`/`assets/`) может потребовать адаптации под их загрузчик файлов.
Формы захвата лидов у ГетКурса обычно свои — возможно, `lead-modal.jsx` стоит
заменить на их виджет вместо кастомного вебхука.

## ⚠️ Что нужно доделать перед запуском

1. **Вставить реальные значения в `lead-modal.jsx`** (сейчас заглушки):
   `LEADS_WEBHOOK_URL` — см. `docs/leads-setup.md` (Google Таблица через Apps Script);
   `REDIRECT_URL` — куда вести человека после отправки формы.
2. **Юр. ссылки в подвале** («#») → оферта, политика, согласие на обработку данных.
3. **Продакшен-хардненинг** (по желанию, для домена): предкомпилировать Babel,
   заменить dev-сборки React на prod-min, самохостить шрифты. Сейчас работает на CDN
   + Babel-in-browser — ок для превью, тяжеловато для боевого домена.

## Заметки по переносу

- Дизайн-панель «Tweaks» (инструмент редактирования в Claude Design) из
  `index.html` убрана, её значения зашиты: акцент coral, энергия handheld, видео-герой.
- Картинки в `assets/` получены не через MCP (там жёсткий лимит 256 КБ на файл), а из
  zip-архива с полным экспортом проекта Claude Design — так дошли в полном разрешении.
  Обращения к `window.__resources[...]` (служебный global редактора) заменены на
  обычные `<img src="assets/...">`.
- Флагман «Большая Игра» в этом дизайне **анонсируется** (карточка дня 4 + оффер) —
  это отличается от раннего брифа (`content/source-brief.md`), где договаривались флагман
  на лендинге не называть. Решение по актуальному дизайну — за тобой.
- Папки `docs/` и `styles/` (tokens.css, base.css) — ранняя проработка направления
  «премиум-минимал», которое **не пошло в дело**. Актуальная дизайн-система — `tokens/` +
  `_ds_bundle.js`. Старые файлы можно удалить (спроси, если нужно).
